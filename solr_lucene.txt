machine:                     m1.xlarge
data location:               local file system


- You must define a schema in conf/schema.xml for the data you wish to index
- Solr can index csv from local disk just fine (you have to enable remote streaming in conf/solrconfig.xml)
- Command:
     curl 'http://localhost:8983/solr/update/csv?stream.file=exampledocs/books.csv&stream.contentType=text/plain;charset=utf-8&fieldnames=id,cat,name,,inStock,author_t,series_t,,genre_s&commit=true'
- Command to index tweets (these fields MUST be defined in the schema.xml file):
     curl 'http://localhost:8983/solr/update/csv?commit=true&separator=%09&escape=\&stream.file=exampledocs/tweet.tsv&stream.contentType=text/plain;charset=utf-8&fieldnames=,id,created_at,user_id,screen_name,,,in_reply_to_screen_name,,in_reply_to_status_id,text,source,,latitude,longitude,,'


- Set the <dataDir> in solrconfig.xml to reasonable place
